📰 Sources

🎯 Relevance Rules

🔑 Where the keys live
1
Gemini, Feedly, and email keys are set in Netlify — not in this page
Go to your Netlify site → Site settings → Environment variables and add: GEMINI_API_KEY, FEEDLY_ACCESS_TOKEN, FEEDLY_STREAM_ID, RESEND_API_KEY, DIGEST_TO_EMAIL, DIGEST_FROM_EMAIL. Full steps are in the README that ships with this project.
2
Why this way
Anything typed into a browser page is visible to anyone who opens developer tools. Netlify environment variables are only readable by your own server-side functions, so this is the version of the app that's actually safe to hand to a client.
⏰ Automatic daily schedule
Set in netlify.toml: the scheduled-digest function runs automatically at 0 9 * * * (9:00am UTC) every day and emails the high-relevance articles. Change the cron string in that file to move the time — no code changes needed elsewhere.
